# BadBunsy-Bet

BadBunsy-Bet es una plataforma de apuestas privadas desarrollada para hacer más entretenidas las partidas entre amigos en el modo Clubes Pro de EA SPORTS FC.

El proyecto fue creado inicialmente para un grupo privado de amigos del club Bad Bunsy, permitiendo realizar apuestas con una moneda virtual, gestionar partidos, eventos personalizados, rankings, estadísticas y transferencias entre usuarios.

## Características

* Sistema de apuestas simples y combinadas.
* Moneda virtual BB (Bad Bunsy Coins).
* Panel de administración completo.
* Gestión de partidos y eventos.
* Ranking de usuarios.
* Historial de apuestas.
* Transferencias de saldo entre usuarios.
* Configuración avanzada del sistema.
* Restricciones de apuestas por jugador asociado.
* Estadísticas y seguimiento de resultados.

## Objetivo

El objetivo de este proyecto es añadir una capa extra de diversión y competitividad a las partidas entre amigos, sin involucrar dinero real ni fines comerciales.

## Personalización

Aunque fue diseñado específicamente para nuestro grupo de amigos y nuestras necesidades dentro de Clubes Pro, el sistema puede ser adaptado fácilmente para cualquier comunidad, torneo privado o grupo de jugadores que desee implementar un sistema de apuestas virtuales personalizado.

## Tecnologías utilizadas

* PHP
* MySQL
* JavaScript
* HTML5
* CSS3

## Aviso

Este proyecto fue creado únicamente con fines recreativos y de entretenimiento. No está destinado al uso de apuestas con dinero real ni a actividades de juego reguladas.
